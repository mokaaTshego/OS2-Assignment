How to run the program:
	1. Go to the molecule directory where there's makefile,src and bin folders.
	2. Open in terminal and using the makefile to clean type "make clean"
	3. To Compile the code using the makefile type "make" on the terimnal.
	3. To run the program use:
		3.1. "make run" -> java -cp bin molecule.RunSimulation 24 9
		3.2. "make run1" -> java -cp bin molecule.RunSimulation 4 12
		3.3. "make run2" -> java -cp bin molecule.RunSimulation 8 2
